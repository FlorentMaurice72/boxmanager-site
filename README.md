# BoxManager — Site vitrine

Site vitrine statique pour [BoxManager](https://boxmanager.fr).

## Structure

```
├── index.html   # Page principale
├── vercel.json  # Configuration Vercel
└── README.md
```

## Déploiement

Le site est déployé automatiquement sur **Vercel** à chaque push sur `main`.

## Modifier le site

```bash
# Modifier index.html, puis :
git add index.html
git commit -m "Mise à jour du site"
git push
```

Vercel redéploie en moins d'une minute.
